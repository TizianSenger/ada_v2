Drop optional WAV files in this folder to enable splash console sound effects.

Supported file names:
- boot-key.wav        # short key/click sound while log lines stream
- boot-complete.wav   # played once when all boot checks are done

Notes:
- Missing files are handled gracefully (no errors shown).
- Keep clips very short for a responsive startup feel.
- Recommended sample rate: 44.1 kHz or 48 kHz.
